# Sistema de Gerenciamento de Biblioteca 📚

Este é um projeto mais robusto e completo que desenvolvi em PHP utilizando **Programação Orientada a Objetos (POO)** e arquitetura com separação de responsabilidades durante meus estudos no IF.

O sistema foi desenhado para ser um CRUD (Criação, Leitura, Atualização e Deleção) completo conectado a um banco de dados.

## 🛠️ Tecnologias e Conceitos Aplicados:
*   **Backend:** PHP estruturado com Classes, Métodos e Objetos.
*   **Autoloading:** Uso de um script `autoload.php` para carregamento automático de classes, evitando múltiplos `include` manuais.
*   **Interface Visual:** Desenvolvida com HTML5 e estilizada usando o framework **Bootstrap** (CSS/JS) para garantir um layout responsivo e moderno.
*   **Banco de Dados:** Preparado para integração e comunicação com uma base de dados **MySQL**.

## 📁 Estrutura do Projeto:
*   `/css` e `/js`: Arquivos de estilização e comportamento do Bootstrap.
*   `/model`: Classes que representam as entidades do sistema (Usuário, Autor, Editora, Livro, Empréstimo) e a lógica de negócios.
*   `/views`: Telas e interfaces que o usuário final interage.
*   `index.html`: Ponto de entrada do aplicativo (contendo a estrutura do Navbar e a página inicial).
